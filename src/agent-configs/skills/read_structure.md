# Skill: Read Structure

Sử dụng công cụ này để hiểu sơ đồ thư mục của dự án hiện tại.

- Mục đích: Tránh ghi đè file quan trọng, biết vị trí các component/style để import chính xác.
- Tham số `path`: Mặc định là "." (thư mục hiện tại của app).
- Lưu ý: Hệ thống sẽ tự động lọc bỏ `node_modules` để kết quả gọn gàng.
